## Feature 功能

- SQL查询
    - 查询工单 
    - 导出
    - 自动补全，智能提示 
    - 查询语句审计
- SQL审核
    - 流程化工单
    - SQL语句检测与执行
    - SQL回滚
    - 历史审核记录
- 推送
    - E-mail工单推送
    - 钉钉webhook机器人工单推送
- 用户权限及管理
    - 角色划分
    - 基于用户的细粒度权限
    - 注册
- 其他
    - todoList
    - LDAP登录  
    - 动态审核规则配置 
- AutoTask自动执行
