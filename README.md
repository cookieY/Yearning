## Feature 功能

- SQL 查询
  - 查询工单
  - 导出
  - 自动补全，智能提示
  - 查询语句审计
- SQL 审核
  - 流程化工单
  - SQL 语句检测与执行
  - SQL 回滚
  - 历史审核记录
- 推送
  - E-mail 工单推送
  - 钉钉 webhook 机器人工单推送
- 用户权限及管理
  - 角色划分
  - 基于用户的细粒度权限
  - 注册
- 其他
  - todoList
  - LDAP 登录
  - 动态审核规则配置
- AutoTask 自动执行
